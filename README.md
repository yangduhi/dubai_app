# Dubai Cookie Finder

GPS 기반 두바이 쫀득 쿠키 재고 확인 Android 앱

## 프로젝트 구조
```
dubai_app/
├── docs/           # AI 작업 지시서 (task_XX.md)
├── scripts/        # Python 유틸리티
├── android/        # Android Studio 프로젝트
└── .agent/         # 워크플로우 정의
```

## 기술 스택
- **Language**: Kotlin
- **UI**: Jetpack Compose
- **Architecture**: MVVM
- **Maps**: Google Maps SDK

## 시작하기
1. `docs/task_01_project_setup.md` 파일 참조
2. Android Studio에서 `android/` 폴더 열기
3. Gradle Sync 후 실행
